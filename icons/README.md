# Icons Directory

This directory should contain the following icon files:
- `icon16.png` (16x16 pixels)
- `icon48.png` (48x48 pixels)
- `icon128.png` (128x128 pixels)

These icons will be displayed in the browser toolbar and extension management pages.

**Note:** You need to add your own icon files here. The extension requires these icons to load properly. You can create simple PNG icons or use an icon generator tool.
